Tentang Proyek

Latar Belakang & Tujuan

MBG Vision adalah prototipe sistem deteksi benda asing pada makanan yang dikembangkan sebagai proyek tugas akhir perkuliahan. Proyek ini mencoba menjawab tantangan pengawasan kualitas pada program Makan Bergizi Gratis (MBG) dengan memanfaatkan model object detection YOLOv8 untuk mengenali objek anomali pada citra ompreng secara otomatis.

✓ Model dilatih menggunakan dataset yang didapatkan dari internet sebagai referensi
✓ Backend dibangun dengan FastAPI, mendukung mode upload dan streaming
✓ Dikembangkan untuk keperluan riset dan demonstrasi, bukan produk komersial

Fitur Unggulan

Semua yang kamu butuhkan

Unggah Instan

Cukup drag & drop foto ompreng dengan pencahayaan terang untuk dianalisis oleh sistem secara otomatis.

Proses Real-time

Tanpa jeda panjang. Deteksi benda asing dilakukan dalam hitungan detik menggunakan backend FastAPI.

Hasil Akurat

Mendapatkan bounding box langsung pada area yang terindikasi terdapat benda asing atau anomali visual.

Keunggulan Teknis

Apa yang Membuat Prototipe Ini Berbeda

01

Dua Mode Deteksi

Mendukung deteksi gambar statis (upload) maupun streaming real-time lewat kamera via WebSocket.

02

Ringan & Mudah Dijalankan

Cukup browser dan webcam standar, tanpa perangkat keras khusus untuk demo.

03

Confidence Score

Setiap hasil deteksi menampilkan skor kepercayaan model agar evaluasi lebih transparan.

04

Arsitektur API Terstruktur

Backend FastAPI dirancang dengan endpoint yang modular sehingga mudah dikembangkan, diuji, dan diintegrasikan dengan model YOLOv8 maupun fitur tambahan di masa depan.

Metodologi

Alur Kerja Sistem

Gambar ompreng diambil melalui upload file atau langsung dari kamera perangkat pengguna.

1
Input Citra
Upload / Kamera

Citra dikirim ke backend FastAPI, lalu diproses oleh model YOLOv8 yang telah dilatih.

2
Inferensi Model
YOLOv8 + FastAPI

Hasil deteksi berupa bounding box dan confidence score dikembalikan dan ditampilkan ke pengguna.

3
Output & Evaluasi
Bounding Box + Skor
